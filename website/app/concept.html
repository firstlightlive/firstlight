CORE FEATURES
⚔
PLEDGE SYSTEM
Users join a group and pledge a financial amount (e.g. ₹15,000/month). Must update daily status — run, gym, steps, proof photo. Verified via Apple Health / Google Fit / Strava API.
⚠
PENALTY ENGINE
Miss a day without proof = auto-debit from wallet or autopay. 3 consecutive misses = removal from group. Rejoin fee = 2x original pledge. Penalty goes to group's charity pool — group votes on where it goes.
★
RANKING & LEADERBOARD
Daily streak tracking, consistency score, group rankings. Badges for milestones (7-day, 30-day, 100-day, 365-day). Rank drops on missed days — public accountability.
💳
WALLET + AUTOPAY
Two ways to fund penalties:
OPTION A — WALLET: User adds money upfront (like Paytm wallet). Penalties auto-deduct from balance. Refundable if they leave clean.
OPTION B — AUTOPAY: User sets up UPI autopay / credit card e-mandate via Razorpay. Penalties deducted automatically on miss. RBI-compliant recurring mandate (max ₹15,000/cycle).
No active wallet + no autopay = cannot join a pledged group.
⚖
GROUP GOVERNANCE — CHARITY POOL
All penalty money goes to a group charity pool. Group members vote monthly on where funds go: registered charities, child welfare NGOs, community events, participant meetups. Transparent ledger — every rupee tracked. Group admin proposes, members vote. 51% majority decides. Receipts shared publicly.
🔒
ORGANIZATION SYSTEM
Create orgs with admins. Multiple groups per org. Custom rules per group (activity type, frequency, penalty amount). Admin dashboard with analytics.
🌐
INDIA FIRST → GLOBAL
Launch in India with Razorpay (UPI autopay + wallet). Expand globally with Stripe (40+ countries). 150+ wearable integrations via Terra API. Multi-currency, multi-language. Prove the model in Bangalore, then scale.

THE BEAUTY: YOU MAKE MORE MONEY WHEN PEOPLE FAIL.
BUT THE CHARITY MODEL MAKES IT FEEL GOOD — "I FAILED, BUT A CHILD GOT HELPED."
THIS IS WHY USERS WON'T HATE THE PENALTY. THEY'LL RESPECT IT.
NEVER DO THIS
- Never take more than 10% of penalties (users will revolt)
- Never make the charity pool opaque (trust = everything)
- Never charge unfairly — Pact died for this exact reason
- Never keep penalty money if verification fails (refund instantly)
