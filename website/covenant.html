RACE DAY CLAUSE
Applies to Ultra Marathon, Marathon, Half Marathon, and any registered race that starts before 6:00 AM
BIB number must be uploaded to the Races page and posted on Instagram before race day as public proof of registration
The race itself counts as the day's run — no separate morning run required
Instagram carousel and reel may be delayed on race day — the race takes priority
If the race starts after 6:00 AM — the regular 5 km morning run must still happen before the race. No exceptions.
Running is never excused. Only the post timing gets flexibility. The run happens — either on the race course or before it.
FLIGHT TRAVEL CLAUSE — RED-EYE WINDOW
If flight departs between 00:30 AM and 06:00 AM — the morning run window is physically impossible
Upon landing, walk 5 km from the airport — GPS verified via Strava. No cab until the walk is done.
The walk must happen immediately after landing — no hotel first, no delays
Instagram post can be delayed until after the walk is complete and uploaded
